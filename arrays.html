Array Search Methods

  1. Array indexOf()
    The indexOf method returns the first index at which a given element can be found in the array, or -1 if it is not present.
  2. Array lastIndexOf()
    The lastIndexOf method returns the last index at which a given element can be found in the array, or -1 if it is not present.
  3. Array includes()
    The includes method determines whether an array includes a certain value among its entries, returning true or false.
  4. Array find()
    The find method returns the value of the first element in the array that satisfies the provided testing function. Otherwise, it returns undefined.
  5. Array findIndex()
    The findIndex method returns the index of the first element in the array that satisfies the provided testing function. Otherwise, it returns -1.
  6. Array findLast()
    the findLast() method that will start from the end of an array and return the value of the first element that satisfies a condition.
  7. Array findLastIndex()
    The findLastIndex() method finds the index of the last element that satisfies a condition.

JavaScript Array Iteration

  1. Array forEach
    The forEach method executes a provided function once for each array element. It doesnot return any value, if we try to return it will result undefined. Also changes the original array. Since forEach returns undefined, it cannot be chained with other array methods directly.
  2. Array map()
    The map method creates a new array populated with the results of calling a provided function on every element in the calling array. map returns a new array, so it can be chained with other array methods (e.g., filter, reduce).
  3. Array filter()
    The filter method creates a new array with all elements that pass the test implemented by the provided function.
  4. Array reduce()
    The reduce method executes a reducer function (that you provide) on each element of the array, resulting in a single output value.
  5. Array reduceRight()
    The reduceRight method applies a function against an accumulator and each value of the array (from right-to-left) to reduce it to a single value.
  6. Array every()
    The every method tests whether all elements in the array pass the test implemented by the provided function. It returns a Boolean value.
  7. Array some()
    The some method tests whether at least one element in the array passes the test implemented by the provided function. It returns a Boolean value.
  8. Array keys()
    The keys method returns a new Array Iterator object that contains the keys for each index in the array.
  9. Array values()
    The values method returns a new Array Iterator object that contains the values for each index in the array.
  10. Array entries()
    The entries method returns a new Array Iterator object that contains the key/value pairs for each index in the array.
