Object Methods

  1. Object.assign(target, source)
    When you use Object.assign(person1, person2), it copies all enumerable own properties from person2 to person1. This means that the properties in person2 will overwrite the properties with the same key in person1.
  2. Object.create(object)
  3. Object.entries(object)
    Returns an array of a given object's own enumerable string-keyed property [key, value] pairs.
  4. Object.fromEntries()
    The fromEntries() method creates an object from a list of key/value pairs.
  5. Object.keys(object)
    The Object.keys() method returns an array with the keys of an object.
  6. Object.values(object)
    Object.values() is similar to Object.entries(), but returns a single dimension array of the object values
  7. Object.groupBy(object, callback)
