JavaScript Exercises

  1. Write a JavaScript program to check two numbers and return true if one of the number is 100 or if the sum of the two numbers is 100
  2. Write a JavaScript program to get the extension of file name
  3. Write a JavaScript program to replace every character in a given string with the character following it in the alphabet.
  4. Write a JavaScript program to get the current date.
  5. Write a JavaScript program to create a new string adding "New!" in front of a given string. If the given string begins with "New!" already then return the original array.
  6. Write a JavaScript program to create a new string from a given string taking the first 3 characters and the last 3 characters of a string and adding them together. The string length must be 3 or more, if not, the original string is returned.
  7. Write a JavaScript prgoram to extract the first half of a string of even length
  8. Write a JavaScript program to concatenate two string except their first character
  9. Given two values, write a JavaScript program to find out which one is nearest to 100
